### 使用简介
1. 第一个按钮为对话功能，使用此功能可为小精灵下达命令。支持命令见下文。
2. 第二个按钮为窗口切换功能。目前支持的窗口见下文。
3. 最后一个按钮为退出功能。

### 支持命令
1. memo xxx ： 在备忘录界面添加待做的xxx事件。
